#  World Layoffs Analytics Dashboard  
### Power BI + MySQL Data Analytics Project

An end-to-end data analytics project focused on analyzing global layoff trends using **MySQL** and **Power BI**.  
This project demonstrates the complete analytics workflow — from raw data cleaning and SQL-based exploration to dashboard creation and KPI reporting.

---

#  Project Overview

The goal of this project is to uncover patterns and business insights from worldwide layoff data, including:

- Companies with the highest layoffs
- Industry-wise impact
- Funding vs layoffs relationship
- Layoff trends across years
- Company stages affected the most

The project combines:
- **SQL for ETL + EDA**
- **Power BI for visualization and reporting**
- **DAX for KPI calculations**

#  Dataset Information

- **Dataset:** World Layoffs Dataset
- **Source:** Kaggle
- **Format:** CSV
- **Database Table:** `layoffs`

The dataset contains information related to:
- Company names
- Industries
- Total layoffs
- Percentage laid off
- Funding raised
- Company stage
- Country
- Dates
